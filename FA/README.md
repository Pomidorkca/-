# Random Password Generator 🔐

**Random Password Generator** — это удобное GUI-приложение для создания надежных случайных паролей с возможностью сохранения истории генераций.

## 👤 Информация об авторе
* **Имя и фамилия:** Смоляков Егор

## 📝 Описание программы
Приложение написано на языке Python с использованием встроенной библиотеки `tkinter` для графического интерфейса. Оно позволяет пользователю гибко настраивать параметры будущего пароля и сохранять понравившиеся варианты в локальную базу данных (формат JSON), чтобы не потерять их после закрытия программы.

### ✨ Возможности:
* Выбор длины пароля с помощью удобного ползунка (от 4 до 32 символов).
* Настройка набора символов через чекбоксы:
  * Цифры (`0-9`)
  * Буквы английского алфавита (`a-z`)
  * Специальные символы (`_`, `*`, `$`, `%`)
* Защита от ошибок ввода (программа предупредит, если не выбран ни один тип символов).
* Сохранение сгенерированных паролей в файл `passwords.json`.
* Просмотр сохраненной истории прямо в окне приложения.
* Полная очистка истории в один клик.

## 🚀 Инструкция и примеры использования

**Запуск приложения:**
1. Убедитесь, что у вас установлен Python.
2. Запустите файл `main.py` через среду разработки (например, PyCharm) или двойным кликом.

**Как сгенерировать пароль:**
1. В главном окне выставьте ползунком нужную длину (например, `16`).
2. Отметьте галочками нужные типы символов (например, оставить только буквы и цифры).
3. Нажмите кнопку **"Сгенерировать"**. Кнопка загорится зеленым, а пароль появится на экране.

**Как работать с историей:**
* **Сохранить:** После генерации пароля нажмите *"Сохранить этот пароль в историю"*. Приложение запишет его в файл `passwords.json`.
* **Посмотреть:** Нажмите *"Посмотреть историю"*, и все ранее сохраненные пароли выведутся списком в нижней части окна.
* **Очистить:** Если пароли больше не нужны, нажмите *"Очистить историю"* — база данных будет обнулена.
